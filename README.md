
# Работа с API вакансий: HeadHunter и SuperJob

Этот проект представляет собой парсер данных о вакансиях с сайтов HeadHunter (hh.ru) и SuperJob (superjob.ru), а также инструмент для расчета средней зарплаты по выбранным профессиям. Парсер собирает информацию о вакансиях, их зарплатах и сохраняет статистику по каждому запросу.

## Описание

Программа собирает информацию о вакансиях с двух популярных сайтов по поиску работы в России — HeadHunter и SuperJob. На основе этих данных она вычисляет среднюю зарплату по профессиям, и выводит статистику в виде таблицы.

### Основные функции:

1. **Получение данных с HeadHunter (hh.ru) и SuperJob (superjob.ru)**:
    - Получает список вакансий по указанным профессиям.
    - Извлекает зарплаты из вакансий, если они указаны.
    - Рассчитывает среднюю зарплату по профессиям.

2. **Вывод статистики**:
    - Статистика по каждой профессии выводится в табличном формате.
    - Для каждой профессии указывается:
        - Количество найденных вакансий.
        - Количество обработанных вакансий.
        - Средняя зарплата по всем вакансиям для данной профессии.

3. **Конфиденциальные данные**:
    - Для работы с API SuperJob требуется ключ, который хранится в `.env` файле.

## Структура проекта

Проект состоит из нескольких Python файлов, каждый из которых отвечает за отдельную функциональность:

- `hh_statistics.py` — сбор данных с HeadHunter.
- `sj_statistics.py` — сбор данных с SuperJob.
- `statistics_table.py` — вывод статистики в виде таблицы.
- `main.py` — основной исполняемый файл, который интегрирует все компоненты и запускает сбор данных и вывод статистики.

## Установка

1. Клонируйте репозиторий:

    ```bash
    git clone https://github.com/your-username/your-repository-name.git
    ```

2. Перейдите в директорию проекта:

    ```bash
    cd your-repository-name
    ```

3. Создайте виртуальное окружение:

    ```bash
    python3 -m venv venv
    ```

4. Активируйте виртуальное окружение:

    - Для Windows:
      ```bash
      venv\Scripts\activate
      ```
    - Для macOS/Linux:
      ```bash
      source venv/bin/activate
      ```

5. Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```

6. Создайте файл `.env` в корневой директории проекта и добавьте ваш ключ для SuperJob:

    ```bash
    SJ_SECRET_KEY=your_superjob_secret_key
    ```

## Использование

1. В файле `main.py` установите профессии, которые хотите исследовать. Пример:

    ```python
    professions = ["python", "c", "java", "js"]
    ```

2. Запустите программу:

    ```bash
    python main.py
    ```

3. После выполнения программы статистика будет выведена в консоль в виде таблицы.

## Пример вывода

```
+--------------------+-------------------+---------------------+-------------------+
| Язык программирования | Вакансий найдено  | Вакансий обработано | Средняя зарплата  |
+--------------------+-------------------+---------------------+-------------------+
| python             | 1000              | 995                 | 100000            |
| java               | 800               | 785                 | 95000             |
+--------------------+-------------------+---------------------+-------------------+
```

## Примечания

- Все запросы к API SuperJob требуют наличия действующего секретного ключа, который вы можете получить на [сайте SuperJob](https://www.superjob.ru/).
- В проекте использованы такие библиотеки как `requests`, `dotenv` для работы с API и `.env` файлом, а также `terminaltables` для вывода таблиц.
